## 使用技術
### フロントエンド
* Vue.js: 2.6.11
* Vue CLI: 4.4.6
* HTML/CSS
### バックエンド
* Python: 3.8.3
* Django: 3.0
* Django REST Framework: 3.10
* poetry: 1.0.10

## 主な機能
### SPA
* VueとDjango REST Frameworkを使用することでフロントエンドとバックエンドを分離
* Vue CLIでbuildした静的ファイルをS3へデプロイ、CloudFrontで配信
